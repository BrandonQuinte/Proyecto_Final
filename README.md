# Todo List Fullstack — Proyecto Completo

Proyecto base para la Actividad Final Integradora.

Incluye:
- Frontend: React (Vite)
- Backend: Express + MongoDB
- Base de datos: Railway
- Deploy: Vercel + Render
- CI/CD con GitHub Actions

## Instalación

### Frontend
```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

### Backend
```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

## Deploy
- Conectar frontend → Vercel
- Conectar backend → Render
- Base de datos → Railway
